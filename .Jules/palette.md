## 2025-01-01 - Initial Setup\n**Learning:** This is a vanilla HTML/CSS/JS site, no package manager (no pnpm, npm, etc.)\n**Action:** No build step to run, modifications are directly to source HTML/CSS/JS files.
## 2025-01-01 - Missing Skip Link\n**Learning:** This site does not have a 'skip to content' link for keyboard users. This is a common and critical accessibility feature for screen readers and keyboard navigation.\n**Action:** Add a 'skip to content' link to the top of every page and style it so it is visually hidden until focused.
## 2025-01-01 - Target Blank Vulnerability / Accessibility\n**Learning:** Using `target="_blank"` without `rel="noopener noreferrer"` can be a security vulnerability and also lacks the accessible text warning screen reader users that a link opens in a new tab.\n**Action:** Add `rel="noopener noreferrer"` and `<span class="sr-only"> (opens in a new tab)</span>` to `target="_blank"` links. Since there is no `.sr-only` class, adding a title attribute or `aria-label` could be a UX alternative without new CSS.

# New Leaf Main Site

Modern, premium agency website for **New Leaf**. Built with a focus on strategic design, organic aesthetics, and high-performance development.

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom design tokens, grid/flexbox layouts, and smooth transitions.
- **JavaScript**: Minimal, intentional logic for navigation and interactivity.

## Project Structure

```
newleaf-main/
  src/
    pages/        # HTML content
    assets/       # Images, Icons, Video, Fonts
    styles/       # Modular CSS (Tokens, Base, Components, Page-specific)
    scripts/      # Modular JS
  public/         # Static assets (robots, sitemap)
```

## Getting Started

Simply open `src/pages/index.html` in your browser.

## Design Philosophy

The "New Leaf" brand represents professional growth, freshness, and clarity. The design uses a palette of forest greens, sage, and warm ochre to create a premium, organic feel.
